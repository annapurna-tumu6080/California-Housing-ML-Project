# California-Housing-ML-Project
Developed a California Housing Price Prediction system using Python and Scikit-learn. Conducted EDA, feature selection, model training, and evaluation. Compared Linear Regression and Random Forest models, achieving improved prediction accuracy through ensemble learning.

## Project Overview
This project predicts California housing prices using Machine Learning techniques. The dataset contains housing and demographic information such as median income, house age, population, rooms, bedrooms, latitude, and longitude.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Loading
2. Data Understanding
3. Missing Value Analysis
4. Exploratory Data Analysis (EDA)
5. Feature and Target Separation
6. Train-Test Split
7. Linear Regression Model Training
8. Model Evaluation
9. Visualization
10. Random Forest Regression
11. Model Comparison

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Outcome
Random Forest achieved better performance than Linear Regression, demonstrating its ability to capture complex relationships within housing data.
